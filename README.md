### 🚀 What I'm building right now

**[job.os](https://github.com/hemnaath04/job-os)** — personal AI job-search platform. Application tracker, multi-pass resume tailoring agent, and multi-source discovery (TheirStack + SimplifyJobs intern / new-grad README crawl). FastAPI + Postgres / pgvector + Claude through a Manifest gateway. Core invariant: every bullet in a generated resume traces to a verified profile fact — the agent surfaces gap questions instead of inventing experience.

**BedRocked** — civic infrastructure search tool (Cyvl × Autodesk × NVIDIA Physical-AI Hackathon). Natural-language retrieval over the City of Boston's combined-sewer network, scoped to refuse off-topic queries, plus a 3D dig-plan exporter into the Autodesk Platform Services Viewer. [Live](https://sewershed-bedrocked.vercel.app).

**HackRadar** — global hackathon aggregator (web + iOS) that surfaces deadlines and tags by track. [hackradar-v2.vercel.app](https://hackradar-v2.vercel.app).

**Infant Cry Classifier** — health-AI project: distilled a vision/audio teacher into a deployable student model for cry-type classification with cents-per-inference cost.
